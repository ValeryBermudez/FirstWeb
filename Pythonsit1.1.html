Type Definition tipo Definicion Ejemplo
Integer Represents whole numbers, i.e., numbers without decimals. Entero Representa números enteros, es decir, números sin decimales. 1,2,3,4,5...
Float Represents numbers with decimals. Flotante Representa números con decimales. 0,0033.00
String Represents text or a sequence of characters. Cadena Representa texto o una cadena de secuencia de caracteres. “Hello world”
Boolean Represents truth values, i.e., "True" or "False." Used for decision-making. Booleano Representa valores de verdad, es decir, "Verdadero" o "Falso". Utilizado para la toma de decisiones. if - else, while, true - false
List Stores an ordered collection of elements. Elements can be of different types and are accessed by indices. Lista Almacena una colección ordenada de elementos. Los elementos pueden ser de diferentes tipos y se accede a ellos mediante índices. lista = [1, 2, 3, 4]
